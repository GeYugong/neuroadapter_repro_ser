# E0 Public ROI Mapping

Generated artifacts in this directory use the public Algonauts Project 2023
Subject 1 fsaverage ROI masks as the primary protocol.

The official NSD native-label resampling remains available under the legacy
`experiments/roi_ablation/mapping/` tree as a sensitivity analysis. It is not
used as the primary mapping for the functional ROI causal study.

## Result

All 97 functionally labeled parcels in the full 1000-parcel inventory are
already retained by top-SNR-200. V4, Face, and Word each have a selection rate
of 1.0. The public mapping therefore provides no evidence that top-SNR-200
reduces their coverage.
